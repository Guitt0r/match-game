# match-game
Match game, where user can play with Computer
## Technology stack
- React
- TypeScript
- React Hooks
## Installation
Use [yarn](https://yarnpkg.com/) as your package manager to intall dependencies:
```
yarn
```
## Run
```
yarn dev
```
